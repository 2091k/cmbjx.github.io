安装 Docker 程序

```sh
install-docker.sh
```

### 终端命令添加 Docker 镜像加速
方法 ：
```sh
echo '{
"registry-mirrors": [
"https://proxy.1panel.live",
"https://docker.1panel.top",
"https://docker.m.daocloud.io",
"https://docker.1ms.run",
"https://docker.ketches.cn"
]
}' | sudo tee /etc/docker/daemon.json
```
