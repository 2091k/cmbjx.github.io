#### 用bat脚本 默认是修改pdf文件名称

可以自己按需求修改

```sh
@Echo Off & SetLocal ENABLEDELAYEDEXPANSION
FOR %%a in (*.pdf) do (
    set "name=%%a"
    set "name=!name:(=!"
    set "name=!name:)=!"
    ren "%%a" "!name!"
)
exit
```
