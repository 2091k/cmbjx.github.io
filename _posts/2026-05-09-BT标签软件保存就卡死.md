#### 直接可用的注册表 .reg 文件

复制下面全部内容，新建文本文档，粘贴进去，保存为 BT 修复保存卡死.reg，编码选ANSI，然后双击导入注册表即可。

```reg
Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\SOFTWARE\Seagull Scientific\Librarian\File Dialog]
"Use Windows Dialog"=dword:00000001
```
### 使用步骤

- 桌面右键 → 新建 → 文本文档
- 把上面整段代码粘贴进去
- 文件 → 另存为
- 文件名：BT修复保存卡死.reg
- 保存类型：所有文件
- 编码：ANSI
- 保存后完全关闭 Bartender
- 双击这个 .reg 文件 → 弹窗点是导入成功
- 重新打开 Bartender，再点保存 / 另存为就不会卡死了

原理:把保存界面改成win系统默认保存界面
